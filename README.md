## Описание

В ходе работы я изучила основные структурные единицы языка JavaScript, а также инструменты для работы с ним.

**JavaScript (JS)** — это интерпретируемый язык программирования, который
изначально создавался для работы в браузере и управления поведением веб-страниц.

В JavaScript используются 3 ключевых слова для объявления переменных:

1. `let` - изменяемая переменная
2. `const` - константа
3. `var` - устаревший способ (использовать не рекомендуется)

В лабораторной работе также разбирались типы данных JS:

1. Примитивные (primitive) типы
   К нему относятся: `string`, `number`, `BigInt`, `Boolean`, `Undefined`, `Null`, `Symbol`, `NaN`

Пример кода с примитивными типами:

```
let price = 99.99;
let temperature = -15;
let infinity = 1 / 0;
let notANumber = 0 / 0;
console.log(0.1 + 0.2);
```

2. Объектные (object) типы
   Сюда входят `object`, `array`, `function`, `date`

Пример функции:

```
function sum(a, b) {
     return a + b;
 }

let multiply = function (x, y) {
     return x * y;
 };

console.log(sum(5, 3));
```

Арифметические операции имеют особенность:

```
console.log(10 + "5");
console.log("10" - 5);
```

Константы (const) запрещает переназначение, но не запрещает изменение содержимого!

Проверка типов осуществляется с помощью оператора для определения типов данных **typeof**.

**Особенности**:

- typeof null возвращает "object" (историческая особенность)
- массивы тоже "object"
- функции имеют тип "function"

**Явным преобразование** называется, когда программист сам преобразует тип: Number(), String(), Boolean(), parseInt(), parseFloat(), toString(), +

**Неявное преобразование** - когда JS сам пытается привести тип: "5" + 3 → "53" (конкатенация), "5" - 3 → 2 (число)

**Строгое сравнение** производится с участием оператора `===` - без преобразования типов.

```
console.log(5 === "5");
```

Нестрогое сравнение осуществляется с оператором `==` - JS может привести типы

```
console.log(5 == "5");
```

**Условные операторы в JavaScript** имеют некоторые особенности:

- нет строгой типизации;
- условия могут работать с любыми типами;
- есть строгое и нестрогое сравнение

Пример использования оператора if / else:

```
let yourAge = 18;
if (yourAge >=18) {
    console.log("Доступ разрешён");
} else {
    console.log("Доступ запрещён");
}
```
